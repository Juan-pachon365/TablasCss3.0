Css

Css es un lenguaje de programacion que sirve para mostrar de una mejor manera la informacion en un documento escrito en HTML,Permintiendo modificar elementos como el color,feuntes,margenes,dispocisión,tamaños,animacion Su importancia en el desarrollo es mejora la visualización de la pagina creando una separacion de contenido generando una mejor experiencia al ciente a parte de ayudar a generar una pagina Responsive Design. que genera un mejor rendimiento a la pagina.

 

JavaScript

Es un lengujae de programación dinamico que sirve que es usado principalmente para el desarrollo web junto con HTML y CSS y su importancia en el desarrllo es que JavaScript permite crear sitios interactivos y dinámicos, mejorando la experiencia del usuario. Además, con Node.js, se utiliza tanto en el frontend como en el backend, lo que lo convierte en un lenguaje full-stack. Frameworks como React, Angular y Vue.js facilitan el desarrollo de aplicaciones web avanzadas, mientras que herramientas como React Native permiten crear aplicaciones móviles multiplataforma. Su gran ecosistema y constante evolución lo mantienen relevante y esencial para el desarrollo web y móvil moderno.

 
Nombre del atributo Descripción Valores Posibles Ejemplos de Uso Imagen representativa Fuente de información
font-size Define el tamaño de la fuente del texto dentro de un elemento. Unidades absolutas (px, pt, cm, in): 16px, 12pt Font-size
border-radius Redondea las esquinas de un elemento, como un borde o caja. px: 10px %: 50% (para crear círculos o formas ovaladas) Border-Radius
display Define cómo se debe mostrar un elemento (block, inline, flex, etc.) block, inline, flex, none, inline-block Display
flexbox (display: flex;) Crea un contenedor flexible que distribuye los elementos hijos en una fila o columna, facilitando el diseño responsivo. display: flex;, flex-direction, justify-content, align-items flexbox
position Controla la posición de un elemento en el documento. static, relative, absolute, fixed, sticky Position
box-shadow Aplica una sombra a un elemento (alrededor de su borde). Offset horizontal/vertical (10px 10px), ,Desenfoque (15px) ,Color (rgba(0,0,0,0.5)) box-shadow
overflow Controla qué hacer cuando el contenido de un elemento se desborda fuera de su área asignada. visible, hidden, scroll, auto Overflow
z-index Controla la superposición de elementos en el eje Z (qué elemento se coloca sobre otro). Números enteros: 1, 100, -1 z-index
opacity Define la opacidad (transparencia) de un elemento. Números decimales entre 0 y 1: 0.5, 0.75, 1 (1 es completamente opaco) Opacity
margin y padding margin: Define el espacio exterior de un elemento. padding: Define el espacio interior de un elemento. px, %, em: 10px, 2em, 5% margin and padding
Color El atributo color se utiliza para especificar el color del texto de un elemento. Colores por nombre: red, blue, green, etc. Texto a Color
background-color El atributo background-color define el color de fondo de un elemento. Colores en formato RGB: rgb(255, 255, 0), rgb(173, 216, 230). background-color
text-align El atributo text-align controla la alineación horizontal del texto dentro de su contenedor. left: Alinea el texto a la izquierda. right: Alinea el texto a la derecha. center: Centra el texto. justify: Justifica el texto, distribuyéndolo equitativamente. Text-aliegn
width El atributo width define el ancho de un elemento. Valores en píxeles: 100px, 50px, etc. width
